# JavaInterview

这个仓库是 `interview-desktop` 的题库源仓库。

## 内容

- `docs/`：从本地数据库已同步成功的 JavaGuide 面试题 Markdown 源文档。
- `local/custom_questions.md`：应用内置的自定义面试题。
- `data/interview_questions.json`：当前本地题库的结构化快照，方便备份、审计或二次处理。
- `data/manifest.json`：本次导出的文档清单和题目数量统计。

## 当前导出统计

- GUIDE 文档：25
- Active 题目：573

后续你可以直接在这个仓库里维护 `docs/` 下的 Markdown 文件，然后让桌面应用从本仓库同步。
